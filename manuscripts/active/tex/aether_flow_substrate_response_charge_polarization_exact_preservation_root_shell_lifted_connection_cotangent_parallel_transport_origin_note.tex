The primitive readouts already split the visible object classes, the primitive units are already sections of those readouts, the primitive time reversals already preserve the readouts and units, and the primitive metrics are positive and time-reversal invariant. Restricting to the exact zero shells gives exactly the recorded root objects. Each visible primitive fiber is compact by assumption, and each visible root fiber is its closed intersection with the root-generating shell, so the visible root fibers are compact as well.

Item (3) follows from the restriction formulas \eqref{eq:root_section_from_primitive}. No additional root-level pre-ambient-generating shell data are chosen. The lifted pre-ambient-generating shell maps already live on the deeper primitive spaces, and the recorded root shell sections are simply their restrictions to the root zero shells. Item (4) is the same restriction argument one step later. The recorded lifted ambient-generating shell maps are not inserted separately; they are already present as root-shell restrictions of deeper exact-preserving maps into the same target spaces equipped with the same target metrics.

Items (5) and (6) are the same restriction argument applied first to the lifted shell maps and then to the lifted exact-constraint maps. The recorded root maps are not inserted separately; they are already present as shell restrictions of deeper exact-preserving maps into the same target spaces equipped with the same target metrics.

Item (7) is the cotangent reduction statement. The primitive Liouville one-forms are pulled back from the canonical Liouville forms on the same recorded cotangent bundles, so they are not arbitrary inserted one-forms. Restricting them to the root zero shells gives exactly the recorded root Liouville one-forms, and restricting their exterior derivatives gives exactly the recorded root closed two-forms.

Item (8) follows from ordinary restriction and parallel transport. The primitive skew-Hermitian connection one-forms determine unitary transports on primitive curves, identity on constant curves, multiplicativity under concatenation, and inversion under time reversal. Restricting the one-forms to the root zero shells gives exactly the recorded root connection one-forms, and restricting the primitive transports to curves lying in those zero shells gives exactly the recorded root transports. Thus the current root connection data are not independent curve assignments; they are forced as shell restrictions.

Item (9) is therefore immediate: every geometric component of the recorded root-shell / lifted connection-cotangent / parallel-transport layer is recovered verbatim, but it is no longer primitive.
\end{proof}
