\begin{abstract}
The current active-primary theorem already removed the section-centered residual-normal-form datum \(\PreProtoSectionResidualPackage\) as the primitive stopping point of the active pre-proto-section line by realizing the current benchmark-faithful pre-proto-section dynamics package through a deeper explicit section-centered hidden-response pre-pre-proto-section dynamics package \(\PrePreProtoSectionPackage\). For each sector, the benchmark-facing content of the current hidden-response package collapses to the smaller observer-relevant pullback-quadratic core \(\PrePreProtoSectionPullbackCore\): the compact section-centered displacement body together with the positive pullback operator
\[
\PrePreProtoSectionPullbackOperator{\sigma}(y)

on the section-centered displacement space. The separate hidden-response space, hidden-response body, response-map factorization, and hidden operator no longer carry independent benchmark-facing freedom once that pullback core is fixed. Conversely, every such pullback core admits a canonical hidden-response realization with identity response map. Hence the current hidden-response frontier is not yet minimal: the live burden contracts from the larger hidden-response package to the smaller pullback-quadratic core.

The benchmark boundary remains fixed throughout. The one operative metric is still exactly \(\CompletedMetric\), the ordinary matter route is unchanged, there is no second operative metric, the low-energy matter law is not enlarged, and no stronger promotion language is licensed. \begin{proposition}[Pullback-core properties]
\label{prop:pullback}
For each \(\sigma\in\{\Car,\Cur,\Hom\}\), the pullback-quadratic core of Definition \ref{def:pullback} satisfies:
\begin{enumerate}
    \item \(\PrePreProtoSectionPullbackOperator{\sigma}(y)\) is self-adjoint and
    \[
    \ev{v,\PrePreProtoSectionPullbackOperator{\sigma}(y)v}
    \ge
    \mu_{\sigma}\,\| \PrePreProtoSectionResponseMap{\sigma}(y)v \|^{2}
    >0
    \]
    for every \(y\in\ProtoSectionBody{\sigma}\) and every nonzero \(v\in\PrePreProtoSectionVertical{\sigma}\);
    \item
    \[
    \PrePreProtoSectionPullbackFunctional{\sigma}(y,v)
    =
    \InducedResidualFunctional{\sigma}(y,v)
    \qquad
    \text{for every }(y,v)\in\PrePreProtoSectionResidualBody{\sigma};
    \]
    \item for each fixed \(y\in\ProtoSectionBody{\sigma}\), the unique zero of
    \[
    v\mapsto \PrePreProtoSectionPullbackFunctional{\sigma}(y,v)
    \qquad
    \text{on }
    \PrePreProtoSectionResidualBody{\sigma}(y)
    \]
    is \(v=0\).
\end{enumerate}
\end{proposition}

\begin{proof}
Item (1) follows because \(\PrePreProtoSectionPullbackOperator{\sigma}(y)\) is the adjoint pullback of the positive self-adjoint operator \(\PrePreProtoSectionOperator{\sigma}(y)\). More explicitly,
\[
\ev{v,\PrePreProtoSectionPullbackOperator{\sigma}(y)v}
=
\ev{
\PrePreProtoSectionResponseMap{\sigma}(y)v,
\PrePreProtoSectionOperator{\sigma}(y)
\PrePreProtoSectionResponseMap{\sigma}(y)v
},
\]
which is nonnegative by positivity of \(\PrePreProtoSectionOperator{\sigma}(y)\) and strictly positive for \(v\neq 0\) because \(\PrePreProtoSectionResponseMap{\sigma}(y)\) is injective and \eqref{eq:gap} holds.

Item (2) is exactly the displayed identity above together with Definitions \ref{def:deeppackage} and \ref{def:pullback}. Item (3) follows from item (1), because the quadratic form vanishes if and only if \(\PrePreProtoSectionResponseMap{\sigma}(y)v=0\), which by injectivity is equivalent to \(v=0\).
\end{proof}

\frac{1}{2}
\ev{
\CanonicalResponseMap{\sigma}(y)v,
\CanonicalHiddenOperator{\sigma}(y)
\CanonicalResponseMap{\sigma}(y)v

=
\frac{1}{2}
\ev{v,\PrePreProtoSectionPullbackOperator{\sigma}(y)v}
=
\PrePreProtoSectionPullbackFunctional{\sigma}(y,v).
\]
So the canonical hidden-response realization reproduces exactly the same residual law.
\end{proof}

\section{Main Theorem}

\begin{theorem}[Current hidden-response frontier collapses to the pullback-quadratic core]
\label{thm:main}
Fix the primitive continuation data of Definition \ref{def:fixed}, the proto-section benchmark base and downstream chain of Definition \ref{def:downstream}, and for each sector \(\sigma\in\{\Car,\Cur,\Hom\}\) a benchmark-faithful section-centered hidden-response package in the sense of Definition \ref{def:deeppackage}. Then:
\begin{enumerate}
    \item the current hidden-response package induces an observer-relevant pullback-quadratic core \(\PrePreProtoSectionPullbackCore\) and a benchmark-faithful section-centered residual-normal-form realization through Definition \ref{def:pullback} and Proposition \ref{prop:package};
    \item every such pullback-quadratic core admits a canonical hidden-response realization by Proposition \ref{prop:canonical};
    \item if two benchmark-faithful hidden-response packages induce the same pullback-quadratic core, then they induce the same section-centered residual-normal-form realization and therefore the same current benchmark-faithful pre-proto-section dynamics at current scope;
    \item consequently the current hidden-response frontier is not benchmark-facingly primitive: its live benchmark-facing content collapses to the smaller observer-relevant pullback-quadratic core.
\end{enumerate}
\end{theorem}

\begin{proof}
Item (1) is the content of Definitions \ref{def:pullback} and Proposition \ref{prop:package}. Item (2) is Proposition \ref{prop:canonical}. For item (3), if two hidden-response packages induce the same pullback-quadratic core, then by Definition \ref{def:pullback} they define the same displacement body and the same pullback residual functional. Hence they define the same reconstructed section-centered residual-normal-form realization and the same reconstructed pre-proto-section package.
