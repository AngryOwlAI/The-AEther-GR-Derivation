\begin{abstract}
The current active-primary theorem already removed the full observer-relevant pullback-operator core \(\PullbackOperatorCore\) as the primitive stopping point of the active section-centered hidden-response line by collapsing its benchmark-facing content to the smaller observer-relevant pullback body-operator core \(\PullbackBodyOperatorCore\). The present note takes the honest fallback route available before such a cleaner deeper law is fixed: it proves a strictly smaller theorem-level collapse inside the current body-operator core itself.

The positive result is that, relative to the fixed proto-section benchmark base, the full ambient linear-operator presentation still contains presentational redundancy. What current benchmark-facing downstream uses actually require is only the quadratic pullback response on the recorded section-centered displacement body. Because each displacement fiber has nonempty interior in its recovered ambient span, the body-restricted quadratic response determines the positive self-adjoint pullback operator uniquely. What survives is the smaller observer-relevant pullback body-response core \(\PullbackBodyResponseCore\): the displacement body together with its body-restricted pullback response and benchmark faithfulness.

The benchmark boundary remains fixed throughout. The one operative metric is still exactly \(\CompletedMetric\), the ordinary matter route is unchanged, there is no second operative metric, the low-energy matter law is not enlarged, and no stronger promotion language is licensed. The positive result is therefore a genuine smaller theorem-level collapse strictly inside the current body-operator-core frontier. What remains open is to derive or justify the pullback body-response core from still more primitive explicit substrate structure, or else to prove a still sharper collapse, necessity result, or uniqueness result strictly inside that smaller core.
\end{abstract}

\section{Introduction}

The active derivational program of \TheoryName{} again reaches a point where depth alone is not the honest default move. The current active-primary theorem already showed that the full observer-relevant pullback-operator-core presentation is not the live primitive stopping point at current scope, because the separate ambient displacement-space listing carries no independent benchmark-facing freedom beyond the smaller pullback body-operator core \(\PullbackBodyOperatorCore\). That theorem therefore spent the full operator-core presentation as the live burden.

The present note asks the next narrower question available strictly inside that body-operator core. Once the fixed proto-section benchmark base is held fixed, does the current body-operator core still contain an independently live off-body operator presentation, or is the benchmark-facing content of that operator already fixed by its quadratic response on the actual displacement body? The operator is used benchmark-facingly only through the pullback response that it produces on the recorded displacement body, and because each displacement fiber has nonempty interior, that body response determines the operator uniquely.

\section{Collapse of the Body-Operator Core}

\begin{proposition}[The displacement body still recovers the ambient displacement space]
\label{prop:recover_space}
For every benchmark-faithful pullback body-operator core and every \(\sigma\in\{\Car,\Cur,\Hom\}\),
\[
\RecoveredBodyResponseSpace{\sigma}
=
\PrePreProtoSectionVertical{\sigma}.
\]
\end{proposition}

\begin{proof}
Fix \(y\in\ProtoSectionBody{\sigma}\). By Definition \ref{def:bodyop}(1), the fiber \(\PrePreProtoSectionResidualBody{\sigma}(y)\) has nonempty interior in the ambient finite-dimensional Euclidean space \(\PrePreProtoSectionVertical{\sigma}\). Any subset of a finite-dimensional vector space with nonempty interior spans that vector space. Hence
\[
\operatorname{span}\bigl(\PrePreProtoSectionResidualBody{\sigma}(y)\bigr)
=
\PrePreProtoSectionVertical{\sigma}.
\]
Because that fiber is contained in the union appearing in Definition \ref{def:bodyresp},
\[
\PrePreProtoSectionVertical{\sigma}
\subset
\RecoveredBodyResponseSpace{\sigma}
\subset
\PrePreProtoSectionVertical{\sigma},
\]
which proves the claim.
\end{proof}

\begin{proposition}[The body response determines the pullback operator uniquely]
\label{prop:recover_operator}
For every benchmark-faithful pullback body-operator core and every \(\sigma\in\{\Car,\Cur,\Hom\}\), the body-response core determines at most one positive self-adjoint operator family
\[
\RecoveredBodyResponseOperator{\sigma}(y):
\RecoveredBodyResponseSpace{\sigma}\to\RecoveredBodyResponseSpace{\sigma},
\qquad
y\in\ProtoSectionBody{\sigma},
\]
satisfying
\begin{equation}
\PrePreProtoSectionBodyResponse{\sigma}(y,v)
=
\frac{1}{2}
\ev{v,\RecoveredBodyResponseOperator{\sigma}(y)v}
\qquad
\text{for every }(y,v)\in\PrePreProtoSectionResidualBody{\sigma}.
\label{eq:bodyresp}
\end{equation}
The recorded pullback operator family \(\PrePreProtoSectionPullbackOperator{\sigma}(y)\) is that unique family.
\end{proposition}

\begin{proof}
By Proposition \ref{prop:recover_space}, the recovered ambient space is exactly \(\PrePreProtoSectionVertical{\sigma}\). The recorded operator family \(\PrePreProtoSectionPullbackOperator{\sigma}(y)\) therefore supplies an example satisfying \eqref{eq:bodyresp}, so existence is already in hand for the present body-response core.

To prove uniqueness, fix \(y\in\ProtoSectionBody{\sigma}\) and suppose \(B(y)\) and \(C(y)\) are two positive self-adjoint operators on \(\RecoveredBodyResponseSpace{\sigma}\) satisfying \eqref{eq:bodyresp}. Then
\[
\ev{v,\bigl(B(y)-C(y)\bigr)v}=0
\qquad
\text{for every }
v\in\PrePreProtoSectionResidualBody{\sigma}(y).
\]
The left-hand side is a quadratic polynomial on the finite-dimensional Euclidean space \(\RecoveredBodyResponseSpace{\sigma}\), and by Definition \ref{def:bodyop}(1) the set \(\PrePreProtoSectionResidualBody{\sigma}(y)\) has nonempty interior there. A polynomial that vanishes on a set with nonempty interior vanishes identically. Hence
\[
\ev{v,\bigl(B(y)-C(y)\bigr)v}=0
\qquad
\text{for every }
v\in\RecoveredBodyResponseSpace{\sigma}.
\]
Because \(B(y)-C(y)\) is self-adjoint, polarization gives
\[
\ev{u,\bigl(B(y)-C(y)\bigr)w}
=
\frac{1}{2}
\left(
\ev{u+w,\bigl(B(y)-C(y)\bigr)(u+w)}
-\ev{u,\bigl(B(y)-C(y)\bigr)u}
-\ev{w,\bigl(B(y)-C(y)\bigr)w}
\right)
=0
\]
for all \(u,w\in\RecoveredBodyResponseSpace{\sigma}\). Therefore \(B(y)=C(y)\). Since \(y\) was arbitrary, the operator family is unique and is exactly the recorded pullback operator family.
\end{proof}

The current result does not yet complete a first-principles derivation of GR from substrate microphysics, but it does narrow the live derivational burden one level further on the active line.
