\begin{proposition}[The ambient shell law induces a primitive reversible constrained dynamics normal form]
\label{prop:derive}
Assume Definitions \ref{def:ambient} and \ref{def:canonical}. Then the canonical data of Definition \ref{def:canonical} satisfy every requirement of Definition \ref{def:normalform}.
\end{proposition}

\begin{proof}

\[
\jmath_{\sigma}(X_{\sigma})\times Z_{\sigma}^{\mathrm{cur}}\times Z_{\sigma}^{\mathrm{eq}}
\]
because \(\AmbientRead{\sigma}\) has that property on the shell-plane slice. Equation \eqref{eq:Cdef} gives a linear constraint map and a fixed value. Since \(\HiddenOperator{\sigma}\) is positive self-adjoint with \eqref{eq:hiddengap}, the same is true of \(\NormalOperator{\sigma}\), so \eqref{eq:normgap} holds with \(\mu_{\sigma}=\kappa_{\sigma}^{\mathrm{hid}}\).

The symmetry and time-reversal requirements follow because every element of \(\AmbientSym{\sigma}\) and \(\AmbientTime{\sigma}\) preserves the shell plane, the hidden complement, the reference point, the ambient body, the ambient readout, the ambient shell law, the shell value, and the hidden operator.

For the shell chart, \eqref{eq:chidef} maps \(\ForSpace{\sigma}\) into \(\CoordBody{\sigma}\). Because \(\AmbientEmbed{\sigma}(w)\) lies in \(\AmbientShellLaw{\sigma}^{-1}(\AmbientValue{\sigma})\cap \ShellAffine{\sigma}\), one has
\[

\]
which is \eqref{eq:shellchartcompat}.

For shell solvability, let \(u\in \AffineData{\sigma}(\CoordBody{\sigma})\). Then \(u=\AmbientRead{\sigma}(a)\) for some \(a\in\AmbientBody{\sigma}\cap\ShellAffine{\sigma}\), hence \(u\in\AmbientRead{\sigma}(\AmbientBody{\sigma})\). \]
which proves \eqref{eq:nfsolvability}.

Finally, let \(\delta q\in\ker D\AffineData{\sigma}\) and \(\delta n\in\NormalSpace{\sigma}\). By \eqref{eq:Dhatdef},
\[
D\AffineData{\sigma}(\delta q)
=

Therefore \(\delta q\) satisfies the hypothesis of \eqref{eq:shellcoercive}, while \(\delta n\) is governed by \eqref{eq:hiddengap}. Setting

\kappa_{\sigma}\equiv
\min\{

\]
one obtains

+

\ge
\kappa_{\sigma}
\bigl(

\bigr),
\]
which is \eqref{eq:nfcoercive}.
\end{proof}

\section{Forced Constituents of the Normal Form}

\begin{proposition}[The shell-coordinate and off-shell spaces are forced]
\label{prop:spaces}
Assume Definition \ref{def:ambient}. Then:
\begin{enumerate}
    \item the shell-coordinate space is necessarily the translation space \(\ShellVector{\sigma}\) of the ambient shell plane;
    \item the shell-coordinate body is necessarily the translated shell-plane slice \eqref{eq:Kdef} of the ambient body;
    \item the off-shell space is necessarily the hidden orthogonal complement \(\HiddenSpace{\sigma}\);
    \item the off-shell body is necessarily the ambient hidden projection \eqref{eq:Bdef}.
\end{enumerate}
\end{proposition}

\begin{proof}
The normal form is required to parametrize exactly the ambient shell locus while retaining the shell-plane / hidden-sector splitting of Definition \ref{def:ambient}. Hence its shell-coordinate space must be the translation space of the shell plane. Once \(\AmbientRef{\sigma}\) is fixed, the shell-coordinate body is exactly the set of translated shell-plane points lying in \(\AmbientBody{\sigma}\), which is \eqref{eq:Kdef}. Likewise the hidden sector of the ambient package is, by definition, the orthogonal complement of the shell plane, so no other off-shell space is compatible with the stated ambient splitting. The unique off-shell body compatible with the body-splitting bijection \eqref{eq:splitmap} is the hidden projection \eqref{eq:Bdef}.
\end{proof}

\begin{proposition}[The readout, constraint map/value, and positive operator are forced]
\label{prop:maps}
Assume Definition \ref{def:ambient}. Then:
\begin{enumerate}
    \item the shell-data readout is necessarily the ambient readout restricted to the shell plane and written in translated shell coordinates, namely \eqref{eq:Dhatdef};
    \item the constraint map and constraint value are necessarily the affine shell law on the shell plane written in translated shell coordinates, namely \eqref{eq:Cdef};
    \item the positive off-shell operator is necessarily the hidden-sector gap operator \(\HiddenOperator{\sigma}\).
\end{enumerate}
\end{proposition}

\begin{proof}
Because \eqref{eq:hiddeninvisible} states that the ambient readout ignores hidden directions, the normal form can only recover the ambient data by restricting that readout to the shell plane. Writing that restriction in the translated shell coordinates of \eqref{eq:Kdef} gives \eqref{eq:Dhatdef}; no other shell-data map is compatible with the ambient package.

Likewise \eqref{eq:hiddeninvisible} states that the shell law itself ignores hidden directions. The shell constraint of the normal form must therefore be the shell-plane restriction of \(\AmbientShellLaw{\sigma}\). Since that restriction is affine on the finite-dimensional affine plane \(\ShellAffine{\sigma}\), writing it in translated shell coordinates produces the linear map and fixed value of \eqref{eq:Cdef}. Finally, the positive off-shell operator is the hidden-sector gap operator of Definition \ref{def:ambient}; replacing it would change the hidden quadratic law and hence change the ambient package itself.
\end{proof}
