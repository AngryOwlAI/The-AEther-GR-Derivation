## Primary Closure Rule

A new deepest-layer note is primary only if it does at least one of the following:

- proves a genuine uniqueness, necessity, or inevitability statement for the current deepest layer
- collapses several deeper-origin layers into one sharper theorem
- produces a concrete observer-equation gain toward exact Einstein closure on the same one-metric line

If a proposed note does none of these, treat it as side continuation or archive context rather than the default next step.
