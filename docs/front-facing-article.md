This page introduces the single-entry public article for `The Æther-Flow Interpretation of Relativity`.

The article is a public orientation layer. It does not replace the overview-first exact-closure package as the benchmark deliverable. Instead, it gives first-time readers one coherent document that explains the package, states the claim boundary, and points them into the fixed reading order that the active derivational program must answer to.

- [Flagship article PDF](assets/pdfs/aether_flow_exact_closure_flagship_article.pdf)
- [Flagship article LaTeX source](assets/tex/aether_flow_exact_closure_flagship_article.tex)

## Role Of The Article

- present the exact-closure package in one readable public-facing article
- state clearly that GR is adopted exactly in the active package
- state equally clearly that a completed substrate derivation is not yet claimed
- map readers into the overview and the six core manuscripts
- make clear that the benchmark package now serves as the starting point for an active GR-derivation program

## Reading Guidance

- If you want a single-entry orientation document, read this article first.
- If you want the canonical package front door, go next to [Theory Package](theory-package.md) and begin with the overview.
- If you want the live project status after that, continue to [Active Research](research-archive.md).

## Public Boundary

The existence of a front-facing article should not be read as a shift in the project claim boundary. The active benchmark theory statement remains the overview-first exact-closure package. The article is there to improve accessibility, transparency, and review readiness while the derivational program continues downstream.

This article is included in the first curated public release bundle together with the overview and six core exact-closure manuscripts.
