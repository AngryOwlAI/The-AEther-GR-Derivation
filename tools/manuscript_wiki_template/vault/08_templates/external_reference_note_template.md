# External Reference Note Template

Use this as the structural reference for supplemental literature notes under `02_sources/references/`.

Required sections:

1. Title
2. Reference Status
3. Project Relevance
4. Summary
5. Access
6. Separation Rule
7. Build and Sync Metadata
8. Manual Notes block

External references are supplemental literature only. They must not be used to set `frontline_status`, `project_role`, the benchmark claim boundary, or the active primary burden.
