# Authority And Routing

The local wiki must preserve this order:

1. active `.tex` manuscripts
2. `docs/ACTIVE_MANUSCRIPT_FILE_MAP.csv`
3. built repository PDFs
4. vault notes and indexes

Routing decisions must come from the file map, not from stale prose summaries or backlink counts.

The vault should surface at least:

- routing indexes by `frontline_status`
- index pages by `project_role`
- index pages by `family`
- one page per `line_name`
- concept seed pages for the approved terminology
